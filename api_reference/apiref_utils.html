seaduck.utils.find_ind(array, value, peri=None, ascending=1, above=True)#

Find the index of the nearest value to the given value.

Parameters

  • array (numpy.ndarray) ‚Äì 1D numpy array to search index from

  • value (number) ‚Äì The value to find nearest neighbor with

  • peri (number, optional) ‚Äì The periodicity of the array. For example, 360 for longitude.

  • ascending (int, default 1) ‚Äì Whether the array is in ascending order. 1 for ascending order, -1 for descending order.

  • above (boolean, default True) ‚Äì If True, return the index of the largest item in array smaller than value. Otherwise, return the closest value.

seaduck.utils.find_rel(value, array, darray=None, ascending=1, above=True, peri=None, dx_right=True)#

Find the rel-coords of the 1D coords.

The backend for all find_rel functions

Parameters

  • value (numpy.ndarray) ‚Äì 1D array for the value to find rel-coords.

  • array (numpy.ndarray) ‚Äì The array of potential reference levels.

  • darray (numpy.ndarray, optional) ‚Äì The distances between reference levels.

  • peri (number, optional) ‚Äì The periodicity of the array. For example, 360 for longitude.

  • ascending (int, default 1) ‚Äì Whether the array is in ascending order. 1 for ascending order, -1 for descending order.

  • above (boolean, default True) ‚Äì If True, return the index of the largest item in array smaller than value. Otherwise, return the closest value.

  • dx_right (boolean, default True) ‚Äì If True, darray[i] = abs(array[i+1] - array[i])

Returns

  • ix (numpy.ndarray) ‚Äì Indexes of the reference level

  • rx (numpy.ndarray) ‚Äì Non-dimensional distance to the reference level

  • dx (numpy.ndarray) ‚Äì distance between the reference t level and the next one.

  • bx (numpy.ndarray) ‚Äì Value of the reference level

seaduck.utils.find_rel_z(depth, some_z, some_dz=None, dz_above_z=True)#

Find the rel-coords of the vertical coords.

Parameters

  • depth (numpy.ndarray) ‚Äì 1D array for the depth of interest in meters. More negative means deeper.

  • some_z (numpy.ndarray) ‚Äì The depth of reference depth.

  • some_dz (numpy.ndarray or None) ‚Äì dz_i = abs(z_{i+1}- z_i)

  • dz_above_z (Boolean) ‚Äì Whether the dz as the distance between the depth level and a shallower one(True) or a deeper one(False)

Returns

  • iz (numpy.ndarray) ‚Äì Indexes of the reference z level

  • rz (numpy.ndarray) ‚Äì Non-dimensional distance to the reference z level

  • dz (numpy.ndarray) ‚Äì distance between the reference z level and the next one.
